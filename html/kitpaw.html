KitPaw

Overview

The KitPaw MCU is an adaptation of the open-source pro micro MCU intended for use in space limited environments such as low-profile keyboards. To achieve this, the design of the KitPaw removes most of the on board circuit belonging to the pro micro, including the usb port and led components; leaving only the bare minimum to run as a microcontroller. Following this choice to be as compact as possible, the KitPaw is a smd component, sacrificing ease of use.

The KitPaw uses a ATMega32U4-AU, a easy to program base through arduino or QMK. However, without a usb connection it requires a host to program and function, depending on a usb either on the board it connects to, or with the use of a daughterboard such as the unified daughterboard as it was designed for.
